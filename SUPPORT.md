# Support

## How to get help

**GitHub Discussions** (recommended)
For questions, ideas, and general feedback:
https://github.com/NeuralInverse/neuralinverse/discussions

**GitHub Issues**
For confirmed bugs and feature requests only:
https://github.com/NeuralInverse/neuralinverse/issues

Before opening an issue, please search existing issues and discussions to avoid duplicates.

**Email**
For anything that doesn't fit above:
github@neuralinverse.com

## Before asking for help

1. Press `Cmd+Shift+P` (or `Ctrl+Shift+P` on Windows/Linux) and type `Help: About` - this gives your version and environment info, include it in your question.
2. Check the [HOW_TO_CONTRIBUTE.md](./HOW_TO_CONTRIBUTE.md) for build and setup issues.
3. Search [existing issues](https://github.com/NeuralInverse/neuralinverse/issues) and [discussions](https://github.com/NeuralInverse/neuralinverse/discussions).

## Enterprise Support

For enterprise support, compliance features, and SLAs visit:
https://neuralinverse.com

## Security Issues

Please do not report security vulnerabilities via GitHub Issues. See [SECURITY.md](./SECURITY.md) for responsible disclosure instructions.
